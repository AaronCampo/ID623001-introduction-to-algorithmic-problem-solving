# 03: Introduction to Unity Scripting - Tags and Physics

## Tags

Tags are used to identify and reference Game Objects - like "Player" or "Collectible" - so you can find them and differentiate them easily in code.

Creating tags is pretty straightforward: select **Edit > Project Settings** in the top menu bar, and open **Tags and Layers** in the settings window. Expand the **Tags** list and you'll see two have already been created for you.
